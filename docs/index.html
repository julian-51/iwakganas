Pengumpulan Data

Data Collection

Game Iwak Ganas dibangun dengan prinsip privasi penuh. Kami menegaskan bahwa kami tidak mengumpulkan data pribadi apa pun dari perangkat Anda. Tidak ada pembuatan akun, login media sosial, atau formulir pengisian data di dalam game.

The game Iwak Ganas is built with full privacy principles. We emphasize that we do not collect any personal data from your device. There is no account registration, social media login, or data submission forms inside the game.

Penggunaan Data

Data Usage

Karena kami tidak mengumpulkan data apa pun, maka tidak ada informasi yang kami gunakan, bagikan, atau jual kepada pihak ketiga untuk keperluan apa pun, termasuk iklan atau pelacakan.

Since we do not collect any data, no information is used, shared, or sold to third parties for any purpose, including advertising or tracking.

Penyimpanan Lokal

Local Storage

Game ini menyimpan data tertentu seperti Skor Tertinggi (Highscore) dan Pengaturan Game (Audio/Grafis) secara lokal di penyimpanan internal perangkat Anda. Data ini sepenuhnya berada di bawah kendali Anda dan tidak pernah dikirimkan ke server kami. Data ini akan terhapus otomatis jika Anda menghapus data aplikasi atau mencopot instalasi game.

This game saves certain data such as Highscores and Game Settings (Audio/Graphics) locally on your device's internal storage. This data is entirely under your control and is never transmitted to our servers. This data will be automatically deleted if you clear application data or uninstall the game.

Layanan Pihak Ketiga

Third-Party Services

Kami menggunakan layanan distribusi resmi melalui Google Play Store yang dapat mengumpulkan informasi non-identitas secara anonim untuk laporan kegagalan sistem (crash reports) guna membantu kami memperbaiki bug. Untuk informasi lebih lanjut, silakan pelajari Kebijakan Privasi Google.

We distribute this game through the Google Play Store, which may collect anonymous non-identifiable technical data (such as crash reports) to help us troubleshoot bugs. For more details, please review the Google Privacy Policy.

Batasan Usia

Age Restrictions

Game ini hanya ditujukan untuk pengguna berusia 13 tahun ke atas. Kami tidak secara sengaja menargetkan game ini atau mengumpulkan informasi dari anak-anak di bawah usia 13 tahun. Jika Anda adalah orang tua atau wali yang menemukan bahwa anak Anda di bawah usia 13 tahun menggunakan aplikasi ini, Anda dapat menghapus data aplikasi secara langsung dari pengaturan perangkat atau menghubungi kami.

This game is intended only for users aged 13 and above. We do not knowingly target the game at or collect information from children under the age of 13. If you are a parent or guardian and realize your child under 13 is playing this application, you can delete the application data directly via device settings or contact us.
